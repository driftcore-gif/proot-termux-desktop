🐧 proot-termux-desktop

Linux Desktop on Android

Run Debian, Ubuntu, Arch Linux and desktop environments inside Termux using PRoot. No root required.

GitHub Install

Features

📱 Android

Designed for Android devices.

🔒 Rootless

No root access required.

🖥 Desktop

XFCE, LXQt, MATE and more.

⚡ Fast

Quick setup and deployment.

Installation

pkg update && pkg upgrade 
pkg install git -y

git clone https://github.com/driftcore-gif/proot-termux-desktop.git

cd proot-termux-desktop

bash install.sh
